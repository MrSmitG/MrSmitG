# The Odyssey — Journey Home

A playable browser adventure inspired by Homer’s epic and the spirit of Christopher Nolan’s *The Odyssey*.

## Play

Open [`odyssey/index.html`](odyssey/index.html) in a modern browser, or serve the folder:

```bash
cd odyssey && python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrows | Move Odysseus |
| E / Enter | Talk, interact, advance dialogue |
| Space | Strike with bronze blade |
| Esc | Pause |

## Chapters

1. **The Wine-Dark Sea** — Rally your crew and board the black ship  
2. **The Cave of Polyphemus** — Claim the firebrand and blind the Cyclops  
3. **Circe’s Grove** — Gather moly, free your men, confront the sorceress  
4. **The Sirens’ Song** — Steer the channel while the song pulls you toward the rocks  
5. **The Halls of Ithaca** — Reclaim your bow, clear the suitors, come home to Penelope  

## Stack

Pure HTML, CSS, and Canvas 2D — no build step, no dependencies.
